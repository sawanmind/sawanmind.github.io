## 🚀 Quick Start

```bash

npm install

# Run development server
npm run dev
# → http://localhost:3000

# Build for production
npm run build

# Preview production build
npm run preview

## 📝 Content Management

All portfolio content is managed through a **single source of truth**:

Update this file to change:
- Personal information
- Experience & projects
- Skills & competencies
- Contact information

No code changes needed!

```

## 🔄 Updating Content

```bash
# 1. Edit content
nano public/content.json

# 2. Commit & push

git commit -m "Update portfolio content"
git push

# 3. Auto-deploys in ~2 minutes! ✨
```
